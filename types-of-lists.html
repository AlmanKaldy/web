Description List

Unordered list
это неупорядоченный (маркированный) список, в котором элементы не имеют определённого порядка. Каждый элемент списка обозначается тегом li.

Ordered list
это упорядоченный (нумерованный) список, в котором элементы идут в определённом порядке. Каждый элемент списка обозначается тегом li (list item).

description list
это список описаний, который используется для представления пар "термин — описание". Он состоит из:
  • dl (Description List) — контейнер списка
  • dt (Description Term) — термин
  • dd (Description Definition) — описание термина
